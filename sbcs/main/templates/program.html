Student Experience

Learn how to program

Students learn how to build web applications. No experience, no problem. We cover the fundamental skills needed to get started building your own applications and websites in class.

Build an app

Students will focus on using technology to solve a real world problem. They will be challenged to think beyond basic learning and build an application that has the potential for sustainability and usability.

Visit a major tech company

South Bend Code School's program wraps up with a trip to a major tech company where students present the applications that they have built to a panel of professionals from the tech industry. Past visits have been to companies such as Facebook and Google.

Portfolio of Work

  • Create and publish a biographic website.

  • Work in a team to design, develop, and deploy an online solution to a real world problem.

  • Present their application to an audience of technology professionals.

Code Legion

Students who complete our core program are able to join the Code Legion, an alumni program that consists of additional learning opportunities and experiences.
